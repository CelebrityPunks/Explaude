Welcome to Explaude

Save tweets from your feed. Use them as context in Claude.

1

Connect to Claude

Paste this command into Claude Code. It installs the bridge that lets Explaude save tweets directly to your machine — no popups, no dialogs, fully automatic.

npx explaude setup --id=loading...
Claude will run this for you. Takes about 5 seconds.
2

Restart Chrome

Close and reopen Chrome so it picks up the native messaging bridge.

3

Save any tweet

Right-click any tweet on Twitter/X and select "Save to Explaude". Tweets auto-save to ~/.explaude/tweets.md — Claude can read them anytime.

Opens Twitter/X in a new tab
